# ITA Client (ita-client)

ITA Client

## Install the dependencies

```bash
pnpm i
```

```bash
pnpm dev
```

### Lint the files

```bash
pnpm run lint
```

### Format the files

```bash
pnpm run format
```

### Build the app for production

```bash
pnpm build
```
